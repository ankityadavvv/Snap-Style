# Snap & Style Design Guidelines - Neeman's Branded

## Design Philosophy
A clean, modern aesthetic inspired directly by Neeman's brand - emphasizing sustainability, quality, and simplicity. The design celebrates eco-conscious living through organic typography, green accents, and a minimalist layout that puts products first.

## Core Design Principles
1. **Minimalist & Clean** - Every element serves a purpose, inspired by Neeman's brand simplicity
2. **Eco-Conscious Aesthetic** - Green color palette reflecting sustainable materials and values
3. **Product-Focused** - Large, prominent imagery with minimal distractions
4. **Modern & Professional** - Clean typography and spacious layouts
5. **Mobile-First** - Touch-optimized for seamless mobile experience

## Color Palette

### Primary Colors
- **Primary Green**: Neeman's signature sustainable green (#3D7B4F / hsl(140 34% 43%))
- **Accent Green**: Lighter eco-tone for highlights (#4A9B61 / hsl(140 38% 48%))
- **Neutral White**: Clean background (#FFFFFF)
- **Neutral Gray**: Subtle grays for borders and secondary elements (#F5F5F5 to #EBEBEB)

### Text Colors
- **Primary Text**: Deep charcoal (#1A1A1A)
- **Secondary Text**: Mid gray (#6B7280)
- **Tertiary Text**: Light gray (#9CA3AF)

### Semantic Colors
- **Success**: Green shades
- **Warning**: Amber
- **Destructive**: Red for alerts only

## Typography System

**Font:** DM Sans, Inter (modern, clean sans-serifs)
- Headlines: Bold, generous spacing
- Body: Regular weight for readability
- Emphasis: Medium weight for highlights

**Hierarchy:**
- Hero: text-4xl font-bold
- Section Headers: text-2xl font-semibold
- Card Titles: text-lg font-medium
- Body: text-base font-normal
- Secondary: text-sm font-normal
- Tertiary: text-xs font-normal

## Layout & Spacing

**Spacing System:**
- Compact: p-3 (12px)
- Standard: p-4 (16px)
- Comfortable: p-6 (24px)
- Generous: p-8 (32px)

**Mobile Layout:**
- Max width: max-w-md (448px)
- Horizontal padding: px-4 consistent
- Vertical spacing: space-y-6 to space-y-8

**Cards & Components:**
- Border radius: rounded-lg (8px)
- Shadow: Subtle, minimal - only for floating elements
- Gaps: gap-4 between elements

## Component Styling

### Buttons
- Primary: Green background with white text
- Secondary: Outlined or ghost variants
- State: Subtle hover effect, no color shift
- Size: Generous touch targets (min 44x44px)

### Cards
- Background: White with subtle 1px border
- Border color: Light gray (#E5E7EB)
- Padding: p-5 to p-6
- Shadow: None or very subtle

### Product Cards
- Image: Full-width, rounded corners
- Title: text-lg font-medium
- Price: text-base font-semibold in green
- Badges: Small, muted background with green text
- CTA: Full-width button at bottom

### Forms & Inputs
- Border: 1px light gray
- Focus: Green ring, green border
- Background: White
- Placeholder: Light gray text

## Image Guidelines

**Product Images:**
- Square aspect ratio for consistency
- Clean white or light background
- Professional product photography
- High-quality, sharp details

**Background Images:**
- Minimal use - focus on clean layouts
- Subtle gradients only when needed
- Light, neutral backgrounds

## Animations & Interactions

- Minimal motion for professional feel
- Smooth transitions (200-300ms)
- Focus states clearly visible
- Hover states subtle - use elevate system
- No distracting effects

## Accessibility

- WCAG AA contrast ratios throughout
- Clear focus indicators
- Generous tap targets (44x44px minimum)
- Semantic HTML structure
- Screen reader friendly

## Key UX Flows

1. **Landing** → Clean upload interface with clear value prop
2. **Upload** → Smooth transitions, clear feedback
3. **Analysis** → Loading state with progress indication
4. **Results** → Prominent product display with clear CTAs
5. **Product Links** → Direct to Neeman's website

---

**Design Philosophy:** Snap & Style reflects Neeman's commitment to quality, sustainability, and modern design. The interface should feel as refined as the products themselves - clean, purposeful, and beautiful.
